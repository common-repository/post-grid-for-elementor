== Overview ==

This plugin extend Elementor by adding the Post Grid addon/widget and also the Woocommerce Product Grid widget for free! 

It have several options on Elementor screen to customize each part of the posts grid or Woocommerce product grids.

You also have the option to enable a Isotope filter to your posts or products widget.

It's very useful to displays your posts or Woocommerce products on a specific page, showcase them on the homepage, or even create a archive page for your posts or products with Elementor.

== Features for the Elementor Addons/Widgets ==
* Support for Posts and Products (Woocommerce) post types
* 4 Post/Product Grid Styles - Classic, Masonry and 2 creative styles
* 1,2,3,4 or 5 columns
* Isotope Filter (it only filter the posts that are already displayed on the screen)
* Customize the typography of each item in your post grid
* Customize Borders, Margins and Colors
* Easy to customize
* Post Pagination

== Posts Grid for Elementor Widget ==
With this Elementor widget, you can display the content of hte "post" post type in a customizable post grid, with several style options. 

== Products Grid for Elementor Widget ==
You can use this Elementor widget to display the products of your Woocommerce store.

== Installation Instructions / How to use ==
1. Upload `post-grid-for-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new page using Elementor, or edit a existing open
4. Drag and drop the "Post Grid (PwrGrids)" or "Product Grid (PwrGrids)" Elementor widget to your page.
5. Customize it using the widget options and publish the page.
6. Done!
